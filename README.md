# PyQtPopupDemo


This is a demo Python program of a macOS menu bar extra app that opens a popup.  Although it uses the PySide6 framework, it will also work with PyQt6.

This demo app illustrates how to solve the sporadic issue of the popup window sometimes not being focused when invoked by its menu bar extra icon, causing mouse clicks within it to be ignored or "missed."

PyInstaller builds are available under Releases in the sidebar on the right.  Builds are digitially signed with my Apple Developer certificate and notarized by Apple.
